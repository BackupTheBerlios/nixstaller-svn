What is Nixstaller?

With Nixstaller you can create installers for various UNIX like systems. More details at the about page.

Why Nixstaller?

On most UNIX systems a package manager is used for software installation. If the software you want is not available as a package for your system installation can be tricky. With Nixstaller an installer can be made that will ease the installation process. It also works on many different systems.

What can be installed?

Many kinds of installation procedures can be programmed thanks to the Lua scripting language. Some examples:
  • An installer that extract files to a directory specified by the user.
  • An installer that extracts files to a temporary directory, compiles several binaries and then install these binaries and several other files to the users system.
See also the Vim example installer.

What software license does Nixstaller use?

Nixstaller is released under the terms of GPLv2. Note that this doesn't affect the content used by generated installers. In other words: if you want to modify Nixstaller you have to obey the rules of the GPLv2 license but you can create installers that install files under any software license.
