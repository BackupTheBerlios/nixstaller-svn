What are all these files?

nixstaller-X.Y.tar.gz is a gzipped archive containing all the files.
nixstaller-X.Y.sh is a installer for Nixstaller(works on Linux, FreeBSD and NetBSD).
scons-X.Y.Z-setup.sh is a example installer for SCons (also handy in case you want to compile Nixstaller).
